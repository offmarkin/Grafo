# Grafo

Implementação de grafo capaz de percorrer todas as cidades representadas
em um grafo usando busca em profundidade e busca em largura. Utilizando linguagem de programação Java e interface gráfica Swing. 
A arquitetura foi separada em três partes: dados (o grafo), lógica (os algoritmos BFS e DFS) e a interface gráfica que permite interação com o usuário.
